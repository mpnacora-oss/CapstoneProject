# PC Alley Capstone Project

A multi-branch ERP & POS System built for PC Alley.

## Tech Stack
- **Frontend**: Next.js 14, TailwindCSS, Framer Motion
- **Backend**: Node.js, Express, Sequelize, MySQL

## Getting Started

### Backend
```bash
cd backend
npm install
node src/server.js
```

### Frontend
```bash
cd frontend
npm install
npm run dev -- -p 3002
```
